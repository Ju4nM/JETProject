Jet Climatization

Proposito general

Jet Climatización es un proyecto desarrollado con fines educativos como parte del proyecto integrador que engloba las materias siguientes: Integradora II, Desarrollo Web para I 4.0, Desarrollo Móvil Multiplataforma, Bases de Datos en la Nube y por último pero no menos importante Internet de las cosas. Entre sus funciones tendrá el registro de usuarios, la baja de estos mismos, su inicio de sesión entre otros. Dicha aplicación se desarrollará utilizando las siguientes herramientas: Angular, Xamarin.Forms, C#, BD.
